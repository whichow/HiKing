## 三、商店管理系统

### 1. 刷新机制
- 普通商店：每日0点刷新
- 限时商店：每6小时刷新
- 特惠商店：每周刷新
- 节日商店：根据节日时间

### 2. 折扣系统
- 普通折扣：10%-30%
- 节日折扣：30%-50%
- VIP折扣：额外5%-15%
- 限时折扣：40%-70%

### 3. 库存管理
- 普通商品：无限量
- 限定商品：每日/周限购
- 特殊商品：总量限制
- 活动商品：活动期间限量
