# 《海王人生》游戏策划文档

## 游戏概述
《海王人生》是一款恋爱养成类游戏，玩家通过各种互动方式提升与不同女主角色的好感度。游戏包含剧情模式和挑战模式两种玩法，让玩家体验不同的恋爱攻略乐趣。

## 游戏模式
### 剧情模式
- 按照剧情章节顺序推进
- 每个章节解锁不同女主角色
- 支持多线剧情同时推进
- 包含主线剧情和支线任务

### 挑战模式
- 可同时选择多个女主进行攻略
- 设置攻略时间限制
- 根据完成度获取积分
- 提供排行榜系统
