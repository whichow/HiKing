# 叶澜 - 茶艺师

## 基础设定
### 外貌特征
- 发型：齐肩黑发，偶尔用木簪挽起
- 瞳色：深褐色，如同上等红茶
- 肤色：白皙如玉
- 体型：苗条优雅
- 着装：
  - 工作：改良旗袍式茶服，素雅淡然
  - 私下：现代简约风，偏爱天然面料
  - 约会：中式现代混搭，典雅大方

### 性格特征
- 外在表现：
  - 举止优雅从容，一颦一笑皆有韵味
  - 说话轻声细语，却字字有重量
  - 处事不惊不躁，如同上好的茶香
- 内在性格：
  - 内心坚定，对茶道有独到见解
  - 善于观察，能察觉他人细微情绪
  - 追求完美，但不失人情味
